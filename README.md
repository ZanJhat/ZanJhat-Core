# ZanJhat Core

A core library that provides shared systems such as settings management, UI widgets, utilities, and compatibility tools for ZanJhat mods and other dependent mods.

---

## Features
- Settings management system
- UI widget framework
- Shared utilities & helpers
- Mod compatibility tools
- Dependency base for ZanJhat ecosystem

---

## Compatibility
- API Version: >=1.9.0.2
- SC Version: 2.4.0.0

---

## Usage
ZanJhat Core is required by other ZanJhat mods such as ZanJhat Map.

---

## Installation
1. Install ZanJhat Core first
2. Install dependent mods
3. Ensure correct load order
