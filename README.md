# ZanJhat.Core

A core library that provides shared systems such as settings management, UI widgets, utilities, and compatibility tools for ZanJhat mods and other dependent mods.

## Features

- Centralized settings management system  
- Reusable UI widget components  
- Utility classes and helper functions  
- Shared managers and tools for mod development  
- Compatibility layer for ZanJhat mods and other dependent mods  

## Usage

This mod acts as a dependency for other mods.  
Install **ZanJhat Core** before installing any mods that require it.

## For Mod Developers

You can use the systems provided by ZanJhat Core to:

- Register mod settings  
- Reuse UI widgets  
- Access shared utilities and managers  
- Improve compatibility between mods  

## Compatibility

Designed for mods that depend on ZanJhat Core and other compatible mod projects.
